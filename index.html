So what's special about this project?

Whenever someone visits a site tracked by Google Analytics with javascript disabled browser, the visit is not recorded.

Remy Sharp solved this brilliantly with this WordPress plugin.

This is the .NET porting of that plugin, which enables both ASP.NET MVC and ASP.NET Webforms users to easily add a Google Analytics snippet, which includes a small <NOSCRIPT /> refering an image called __utm.gif loaded from Google's servers.

The request made to that image also includes the tracking parameters for the Google Analytics dashboard:
